Key Features

  • Slice a mesh object along a given axis
  • Cut a mesh object in half along a given axis
  • Mirror a mesh object in half along a given axis
  • Can access the addon from either Object Mode or Edit Mode
  • Supports local or global object location or rotation
  • Sidebar panel can be easily added to any pie menu using Pie Menu Editor Add-on!

Installation

  1. Download the Neltulz - Symmetry zip file anywhere on your computer.
  2. Run Blender.
  3. Go to Edit > Preferences.
  4. In the left sidebar, go to Add-ons.
  5. At the top right, click Install.
  6. Browse for the Neltulz - Symmetry zip file and click Install Add-on from File
    Alternatively, you may double click the zip file to install it.
  7. Next to Generic: Neltulz - Symmetry, enable the checkbox.
  8. At the lower left of the preferences window, click Save Preferences and close the preferences window.

How to Use

Sidebar
  • Access all options from the sidebar in the 3D Viewport

Known Issues & Limitations

  • Applying/Removing modifiers on an object will only remove 1 mirror modifier at a time. Need to add support for multiple mirror modifiers.

Changelog

  • 2020-02-18
    • Added Colorized icons for X,Y,Z (red, green, blue) buttons
    • Added ability to slice, cut and mirror based on user chosen Transform Pivot Location and Orientation
    • Added ability to adjust merge threshold when mirroring.
    • Added Bisect feature when mirroring for extremely accurate mirroring. Eliminates anomalies such as holes and overlapping geometry. Great for mirroring dense objects with lots of polies.
    • Added ability to apply/remove mirror & boolean modifiers and clean up orphaned empties and bisect planes
    • Added ability to Fill with N-gon after performing a cut.
    • Holding CTRL while clicking a mirror button, will mirror the object and keep the mirror and/or bisect modifier
    • Holding CTRL while clicking Apply or Remove will apply or remove the modifiers, but preserve the empties and bisect planes.
  • 2019-12-25 - First Release!
